# Exploratory Data Analyses

A collection of data analysis projects and visualizations.

## Projects

### Turkish Military Chiefs of Staff Birthplace Analysis
Analysis of birthplaces of Turkish Armed Forces Chiefs of Staff during the Republic era.

- **Data Source**: [Turkish Wikipedia](https://tr.wikipedia.org/wiki/T%C3%BCrk_Silahl%C4%B1_Kuvvetleri_genelkurmay_ba%C5%9Fkanlar%C4%B1_listesi)
- **Visualization**: Interactive map showing birthplaces
- **Directory**: `chiefs-of-staff-birthplaces/`

## Setup

```bash
pip install -r requirements.txt
```

## Usage

Navigate to the specific project directory and follow the instructions in its README.
